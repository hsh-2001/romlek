# Romlek Website

Next.js App Router frontend for Romlek.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

The development server starts on `http://localhost:3000`.

## Production

```bash
npm run build
npm run start
```

Set `NEXT_PUBLIC_API_BASE_URL` to point the frontend at a non-local API. It defaults to `http://localhost:4000/api`.
